## 🚀 طريقة الرفع على GitHub Pages

### 1. أنشئ Repository جديد
```
GitHub → New Repository → اسمه: xthon-ar → Public → Create
```

### 2. ارفع الملفين
```
اضغط "uploading an existing file"
اسحب: index.html + video.mp4
اضغط: Commit changes
```

### 3. فعّل GitHub Pages
```
Settings → Pages → Branch: main → / (root) → Save
```

### 4. انتظر دقيقة ثم افتح الرابط
```
https://YOUR-USERNAME.github.io/xthon-ar/
```
